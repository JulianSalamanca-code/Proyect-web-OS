Instalación de Windows

Requisitos para Virtualizar Windows 10/11

Mínimos

  • RAM: 4 GB
  • CPU: 1 Núcleo (vCPU)
  • Disco: 64 GB
  • VRAM: 128 MB

Recomendados

  • RAM: 8 GB
  • CPU: 2 Núcleos o más
  • Disco: 100 GB (SSD)
  • VRAM: 256 MB + 3D

Instalación de Ubuntu

Requisitos para Virtualizar Ubuntu

Mínimos

  • RAM: 2 GB
  • CPU: Dual Core 2GHz
  • Disco: 25 GB
  • Gráficos: VGA

Recomendados

  • RAM: 4 GB
  • CPU: Quad Core 2GHz
  • Disco: 40 GB
  • Gráficos: Aceleración 3D
