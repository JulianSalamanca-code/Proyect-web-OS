Guías de instalación en VM

Aprende paso a paso cómo instalar Windows y Ubuntu en una máquina virtual. Cada vídeo incluye los requisitos mínimos y recomendaciones prácticas.

Instalación de Windows

Requisitos para VM

Mínimos

RAM 4 GB
CPU 1 vCPU
Disco 64 GB
VRAM 128 MB

Recomendados

RAM 8 GB
CPU 2+ vCPU
Disco 100 GB SSD
VRAM 256 MB + 3D

Recomendaciones

  • Activa la virtualización por hardware (VT-x / AMD-V) en la BIOS antes de empezar.
  • Usa un disco SSD como almacenamiento para la VM para evitar tiempos de carga largos.
  • Instala las Guest Additions de VirtualBox al terminar para mejorar resolución y rendimiento.
  • No asignes más del 50% de tu RAM física a la VM para evitar ralentizar el host.
  • Descarga la ISO directamente desde microsoft.com para garantizar que sea legítima.

Instalación de Ubuntu

Requisitos para VM

Mínimos

RAM 2 GB
CPU Dual Core
Disco 25 GB
Gráficos VGA

Recomendados

RAM 4 GB
CPU Quad Core
Disco 40 GB
Gráficos Aceleración 3D

Recomendaciones

  • Elige Instalación mínima durante el setup para un sistema más ligero y rápido en VM.
  • Activa la opción "Descargar actualizaciones al instalar" para tener el sistema al día desde el inicio.
  • Instala VirtualBox Guest Additions para habilitar carpetas compartidas y pantalla completa.
  • Ejecuta sudo apt update && sudo apt upgrade como primer comando tras la instalación.
  • La ISO oficial se descarga en ubuntu.com/download — verifica el checksum SHA256.
