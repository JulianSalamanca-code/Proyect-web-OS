Portal académico de referencia rápida para administración de Windows mediante intérpretes de comandos. Cada sección agrupa comandos por categoría con su descripción, sintaxis de uso y ejemplo de salida real.
CMD — Command Prompt

Intérprete clásico de Windows heredado del sistema MS-DOS. Opera con comandos de texto plano orientados a tareas de administración básica: diagnóstico de disco, reparación de archivos del sistema, información de hardware y automatizaciones simples mediante scripts .bat. Ideal cuando se requiere compatibilidad máxima o un entorno sin dependencias externas.

PowerShell

Shell moderna basada en .NET que devuelve objetos en lugar de texto plano. Permite acceder a WMI/CIM, el registro, servicios, procesos y APIs remotas con un lenguaje de scripting robusto. Módulos, pipelines estructurados y soporte multiplataforma la convierten en la opción estándar para administración avanzada y orquestación a gran escala.
